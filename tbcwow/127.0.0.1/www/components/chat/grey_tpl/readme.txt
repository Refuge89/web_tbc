Installationsanweisung.

1. Entpacken Sie  das Verzeichnis (grey_tpl/) aus der .zip Datei.
2. Legen Sie dieses Verzeichnis samt Inhalt in das ET – Chat Verzeichnis.
3. In der config.php verändern Sie die Template Variable entsprechend:
   $template = "grey_tpl";

Viel Spass!
